# agents

Monorepo of local automation agents.

| Folder | Description |
|--------|-------------|
| [**email-agent/**](email-agent/) | Email-focused agent (backend + frontend). |
| [**nvoids-job-agent/**](nvoids-job-agent/) | Polls [nvoids.com](https://nvoids.com), matches jobs, builds Gmail application drafts. See that folder’s `README.md` for setup and run commands. |

Each subfolder is an independent project with its own dependencies and docs.
